# QR Treasure Hunt (Stateless Version)

An offline-capable web application for scavenging QR codes to collect digital stamps and points.

*Note: This version uses `localStorage` for data persistence. No backend server or database is required.*

## üöÄ Key Features

### 1. QR Scanner (`/scan`)
- Real-time QR recognition using mobile camera (Default: Back Camera)
- **Stateless Verification**: The QR code contains all necessary treasure data (Points, Name, Description).
- Prevents duplicate collection of the same treasure.

### 2. Admin Portal (`/admin`)
- **Access**: Manually navigate to `/admin` (Hidden from UI)
- Create and manage treasures.
- Generate and download QR codes instantly.
- **Note**: Treasures created here are stored locally but the generated QR codes can be scanned by *any* device.

### 3. User Dashboard (`/dashboard`)
- View total points and collected stamps.
- Visualize discovered treasures.

## üõ† Tech Stack

- **Frontend**: Next.js 15 (App Router), Vanilla CSS
- **Storage**: Browser LocalStorage (No Backend)
- **Scanning**: html5-qrcode
- **Generation**: qrcode.react

## üì± QR Code Format

The QR codes generated by this application contain a JSON object with all treasure details. This allows any device to verify the treasure without needing a synchronized database.

- **Content**: JSON String
- **Schema**:
  ```json
  {
    "id": "uuid-v4-string",
    "name": "Treasure Name",
    "points": 50,
    "desc": "Optional description text"
  }
  ```
